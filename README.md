## Run Locally

Clone the project

```bash
$ git clone https://github.com/Singatha/banana-republic.git
```

Go to the banana-republic directory

```bash
$ cd banana-republic
```

Install dependencies

```bash
npm install
```

Start the application

```bash
npm start
```
